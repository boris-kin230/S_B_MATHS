Espaces Vectoriels et Dimension

Cours approfondi : Définitions, Démonstrations et Applications

1. Sous-espace engendré par une famille (Vect)

Définition : Soit $E$ un espace vectoriel sur un corps $\mathbb{K}$ (souvent $\mathbb{R}$ ou $\mathbb{C}$) et $\mathcal{F} = (v_1, \dots, v_p)$ une famille de $p>0$ vecteurs de $E$. On appelle sous-espace engendré par $\mathcal{F}$, noté $Vect(\mathcal{F})$ ou $Vect(v_1, \dots, v_p)$, l'ensemble de toutes les combinaisons linéaires des vecteurs de $\mathcal{F}$.
$$Vect(v_1, \dots, v_p) = \left\{ \sum_{i=1}^p \lambda_i v_i \mid (\lambda_1, \dots, \lambda_p) \in \mathbb{K}^p \right\}$$
Démonstration (C'est un sous-espace vectoriel) :
Pour montrer que $Vect(\mathcal{F})$ est un sous-espace vectoriel (s.e.v.) de $E$, il faut vérifier deux choses :
1. Il est non vide : En choisissant tous les scalaires $\lambda_i = 0$, on obtient $\sum_{i=1}^p 0 \cdot v_i = 0_E$. Donc $0_E \in Vect(\mathcal{F})$, l'ensemble n'est pas vide.
2. Il est stable par combinaison linéaire : Soient $x, y \in Vect(\mathcal{F})$ et $\alpha \in \mathbb{K}$. Par définition, il existe des scalaires $(a_i)$ et $(b_i)$ tels que $x = \sum_{i=1}^p a_i v_i$ et $y = \sum_{i=1}^p b_i v_i$.
Calculons $\alpha x + y = \alpha \left(\sum_{i=1}^p a_i v_i\right) + \sum_{i=1}^p b_i v_i = \sum_{i=1}^p (\alpha a_i + b_i) v_i$.
Puisque $(\alpha a_i + b_i) \in \mathbb{K}$, $\alpha x + y$ s'écrit bien comme une combinaison linéaire des $v_i$. Donc $\alpha x + y \in Vect(\mathcal{F})$.
Conclusion : $Vect(\mathcal{F})$ est bien un s.e.v. de $E$. C'est d'ailleurs le plus petit s.e.v. contenant la famille $\mathcal{F}$.
Application Directe (Mini-Exercice) :
Énoncé : Dans $\mathbb{R}^3$, montrer que le plan $P$ d'équation cartésienne $z = 0$ est un sous-espace vectoriel.
Solution : Un vecteur $(x,y,z) \in P \iff z = 0 \iff (x,y,z) = (x,y,0) = x(1,0,0) + y(0,1,0)$.
On en déduit que $P = Vect((1,0,0), (0,1,0))$. Étant donné qu'il s'écrit comme un Vect, le cours nous affirme (par la démonstration ci-dessus) que $P$ est obligatoirement un sous-espace vectoriel de $\mathbb{R}^3$.

2. Famille Libre de taille maximale

Théorème : Si $\mathcal{F} = (v_1, \dots, v_p)$ est une famille libre d’un espace vectoriel $E$ de dimension $p$ (avec $p>0$), alors $\mathcal{F}$ est une base de $E$.
Démonstration :
Nous savons que $\dim(E) = p$ et que $\mathcal{F}$ possède $p$ éléments et est libre. Pour prouver que c'est une base, il reste à prouver qu'elle est génératrice.
Raisonnement par l'absurde : Supposons que $\mathcal{F}$ n'est pas génératrice. Alors, il existe un vecteur $x \in E$ tel que $x \notin Vect(\mathcal{F})$.
Considérons la famille $\mathcal{F}' = \mathcal{F} \cup \{x\} = (v_1, \dots, v_p, x)$. Montrons qu'elle est libre. Soient des scalaires tels que $\lambda_1 v_1 + \dots + \lambda_p v_p + \mu x = 0_E$.
- Si $\mu \neq 0$, alors $x = -\frac{1}{\mu}(\lambda_1 v_1 + \dots + \lambda_p v_p)$, ce qui signifie que $x \in Vect(\mathcal{F})$. C'est une contradiction.
- Donc $\mu = 0$. Il reste $\lambda_1 v_1 + \dots + \lambda_p v_p = 0_E$. Puisque $\mathcal{F}$ est libre, cela implique $\lambda_1 = \dots = \lambda_p = 0$.
La famille $\mathcal{F}'$ est donc libre. Or, elle contient $p+1$ éléments dans un espace de dimension $p$. Un théorème fondamental affirme que le cardinal de toute famille libre est inférieur ou égal à la dimension de l'espace. Nous avons $p+1 \le p$, ce qui est absurde.
Conclusion : L'hypothèse de départ est fausse, $\mathcal{F}$ est donc génératrice. Libre et génératrice, $\mathcal{F}$ est une base.
Application Directe (Mini-Exercice) :
Énoncé : Les vecteurs $u = (1, 2)$ et $v = (-1, 3)$ forment-ils une base de $\mathbb{R}^2$ ?
Solution : $\mathbb{R}^2$ est de dimension 2. La famille $(u, v)$ contient 2 vecteurs. Montrons qu'elle est libre. On cherche $\lambda, \mu$ tels que $\lambda u + \mu v = 0$.
$\lambda(1) - \mu(1) = 0 \implies \lambda = \mu$
$\lambda(2) + \mu(3) = 0 \implies 2\lambda + 3\lambda = 0 \implies 5\lambda = 0 \implies \lambda = 0$ et $\mu = 0$.
La famille est libre, de cardinal 2 dans un espace de dimension 2, c'est donc automatiquement une base (inutile de vérifier qu'elle est génératrice).

3. Famille Génératrice de taille minimale

Théorème : Si $\mathcal{F} = (v_1, \dots, v_p)$ est une famille génératrice d’un espace vectoriel $E$ de dimension $p$, alors $\mathcal{F}$ est une base de $E$.
Démonstration :
Nous savons que $\dim(E) = p$ et que $\mathcal{F}$ est génératrice de cardinal $p$. Pour prouver que c'est une base, il faut montrer qu'elle est libre.
D'après le théorème de la base incomplète (ou extraction de base), de toute famille génératrice, on peut extraire une base de $E$.
Soit $\mathcal{B}$ la sous-famille extraite de $\mathcal{F}$ qui forme une base de $E$.
Puisque $\mathcal{B}$ est une base de $E$, le nombre d'éléments de $\mathcal{B}$ doit être exactement égal à $\dim(E)$, soit $p$.
Or, $\mathcal{B}$ est incluse dans $\mathcal{F}$ qui possède elle-même $p$ éléments. Le seul moyen pour qu'une partie à $p$ éléments d'un ensemble à $p$ éléments existe est que la partie soit l'ensemble tout entier.
Donc $\mathcal{B} = \mathcal{F}$. Puisque $\mathcal{B}$ est une base (donc libre), $\mathcal{F}$ est libre.
Application Directe (Mini-Exercice) :
Énoncé : Soit $P_2[X]$ l'espace des polynômes de degré $\le 2$. On donne $P_1 = 1, P_2 = X, P_3 = X^2$. Soit $\mathcal{G} = (1+X, 1-X, X^2)$. Montrer que $\mathcal{G}$ est une base.
Solution : $\dim(P_2[X]) = 3$. La famille $\mathcal{G}$ compte 3 éléments. Montrons qu'elle est génératrice : tout polynôme $aX^2 + bX + c$ peut s'écrire avec cette famille.
Remarquons que $X = \frac{1}{2}(1+X) - \frac{1}{2}(1-X)$ et $1 = \frac{1}{2}(1+X) + \frac{1}{2}(1-X)$.
Ainsi, n'importe quel $aX^2 + bX + c$ peut être reconstruit à partir de $\mathcal{G}$. Étant génératrice de cardinal 3 en dimension 3, c'est une base.

4. Intersection de deux sous-espaces vectoriels

Théorème : Soient $F$ et $G$ deux sous-espaces vectoriels d'un espace vectoriel $E$. Alors leur intersection $F \cap G$ est également un sous-espace vectoriel de $E$.
Démonstration :
1. Non vide : $F$ et $G$ étant des s.e.v. de $E$, ils contiennent tous les deux le vecteur nul $0_E$. Par conséquent, $0_E \in F \cap G$. L'intersection n'est pas vide.
2. Stabilité : Soient $x, y \in F \cap G$ et $\alpha \in \mathbb{K}$.
- Comme $x, y \in F$ et que $F$ est un s.e.v., la combinaison $\alpha x + y \in F$.
- Comme $x, y \in G$ et que $G$ est un s.e.v., la combinaison $\alpha x + y \in G$.
Puisque $\alpha x + y$ appartient à la fois à $F$ et à $G$, on a bien $\alpha x + y \in F \cap G$.
Conclusion : $F \cap G$ vérifie les axiomes d'un sous-espace vectoriel. (Attention : l'union $F \cup G$ n'est en général PAS un s.e.v.)
Application Directe (Mini-Exercice) :
Énoncé : Dans $\mathbb{R}^3$, soient les plans vectoriels $F : x + y + z = 0$ et $G : x - y = 0$. Déterminer $F \cap G$.
Solution : Un vecteur $(x,y,z) \in F \cap G$ vérifie le système : $\begin{cases} x + y + z = 0 \\ x - y = 0 \end{cases} \implies \begin{cases} y = x \\ 2x + z = 0 \implies z = -2x \end{cases}$.
Donc $(x,y,z) = (x, x, -2x) = x(1, 1, -2)$. L'intersection est donc $Vect((1, 1, -2))$, qui est bien un sous-espace vectoriel (c'est une droite vectorielle).

5. Formule de Grassmann

Théorème (Formule de Grassmann) : Si $F$ et $G$ sont deux sous-espaces vectoriels de dimension finie d'un espace vectoriel $E$, alors la somme $F+G$ est de dimension finie et :
$$\dim(F + G) = \dim(F) + \dim(G) - \dim(F \cap G)$$
Démonstration :
Soit $k = \dim(F \cap G)$. Soit $(e_1, \dots, e_k)$ une base de $F \cap G$.
Puisque $F \cap G \subset F$, on peut compléter cette base pour obtenir une base de $F$ selon le théorème de la base incomplète : $\mathcal{B}_F = (e_1, \dots, e_k, f_1, \dots, f_p)$. Ainsi $\dim(F) = k + p$.
De même, $F \cap G \subset G$, complétons pour avoir une base de $G$ : $\mathcal{B}_G = (e_1, \dots, e_k, g_1, \dots, g_q)$. Ainsi $\dim(G) = k + q$.
Considérons la famille $\mathcal{B} = (e_1, \dots, e_k, f_1, \dots, f_p, g_1, \dots, g_q)$.
1. Génératrice pour $F+G$ : Tout vecteur de $F+G$ s'écrit $x+y$ avec $x \in F$ et $y \in G$. $x$ est combinaison des $(e_i, f_j)$ et $y$ des $(e_i, g_l)$. Donc $x+y$ est une combinaison linéaire des éléments de $\mathcal{B}$.
2. Libre : Supposons $\sum \alpha_i e_i + \sum \beta_j f_j + \sum \gamma_l g_l = 0_E$.
Alors $\underbrace{\sum \gamma_l g_l}_{\in G} = \underbrace{- \sum \alpha_i e_i - \sum \beta_j f_j}_{\in F}$.
Ce vecteur appartient donc à $F \cap G$. Il s'écrit alors uniquement en fonction des $e_i$. On en déduit que $\sum \gamma_l g_l = \sum \delta_i e_i$, soit $\sum \delta_i e_i - \sum \gamma_l g_l = 0$. Mais $\mathcal{B}_G$ est libre, donc tous les $\gamma_l = 0$ et $\delta_i = 0$.
En reportant $\gamma_l = 0$ dans l'équation initiale, on a $\sum \alpha_i e_i + \sum \beta_j f_j = 0_E$. Or $\mathcal{B}_F$ est libre, donc les $\alpha_i = 0$ et $\beta_j = 0$. La famille $\mathcal{B}$ est bien libre.
Conclusion : $\mathcal{B}$ est une base de $F+G$. Son cardinal est $k + p + q$.
$\dim(F+G) = k + p + q = (k + p) + (k + q) - k = \dim(F) + \dim(G) - \dim(F \cap G)$.
Application Directe (Mini-Exercice) :
Énoncé : Dans $\mathbb{R}^4$, soit $F$ et $G$ deux hyperplans (donc de dimension 3). Quelle est la dimension possible de leur intersection $F \cap G$ si $F \neq G$ ?
Solution : D'après la formule de Grassmann : $\dim(F \cap G) = \dim(F) + \dim(G) - \dim(F + G)$.
Ici, $\dim(F) = \dim(G) = 3$. Puisque $F \neq G$, le sous-espace $F+G$ contient strictement $F$, donc $\dim(F+G) > 3$. Or $F+G$ est un sous-espace de $\mathbb{R}^4$, donc $\dim(F+G) \le 4$. Par conséquent, $\dim(F+G) = 4$.
Ainsi, $\dim(F \cap G) = 3 + 3 - 4 = 2$. L'intersection de deux hyperplans distincts de $\mathbb{R}^4$ est toujours un plan vectoriel de dimension 2.
