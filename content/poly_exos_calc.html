DS Type I. Division Euclidienne & Paramètres
1. Soit $P = (X-2)^{2n} + (X-1)^n - 1$. Montrer que $P$ est divisible par $(X-1)(X-2)$.
2. Déterminer le reste de la division de $X^n$ par $X^2-3X+2$.
3. Application complexe : Trouver le reste de $P = (\cos \theta + X \sin \theta)^n$ par $X^2+1$.
Corrigé détaillé :
1. Évaluons $P$ aux racines du diviseur :
- En $X=1$ : $P(1) = (1-2)^{2n} + (1-1)^n - 1 = (-1)^{2n} - 1 = 1 - 1 = 0$.
- En $X=2$ : $P(2) = (2-2)^{2n} + (2-1)^n - 1 = 1^n - 1 = 0$.
$P$ s'annule en 1 et 2, donc $(X-1)(X-2)$ divise $P$.

2. $X^2-3X+2 = (X-1)(X-2)$. Le reste est de la forme $R = aX + b$.
- $X=1 \implies 1^n = a + b$
- $X=2 \implies 2^n = 2a + b$
Par soustraction : $a = 2^n - 1$ et $b = 1 - (2^n - 1) = 2 - 2^n$.
$R(X) = (2^n - 1)X + (2 - 2^n)$ 3. Les racines de $X^2+1$ sont $i$ et $-i$. Posons $R = aX + b$.
$P(i) = (\cos \theta + i \sin \theta)^n = (e^{i\theta})^n = e^{in\theta} = \cos(n\theta) + i\sin(n\theta)$.
Or $P(i) = R(i) = ai + b$. Par identification :
$R(X) = \sin(n\theta)X + \cos(n\theta)$
Expert II. Polynômes de Tchebychev $T_n$
1. Justifier la relation $T_{n+1} = 2XT_n - T_{n-1}$.
2. Déterminer les racines de $T_n$ et montrer qu'elles sont dans $]-1, 1[$.
3. Étudier la parité de $T_n$.
Analyse :
1. On utilise $\cos(a+b) + \cos(a-b) = 2\cos a \cos b$. Avec $a = n\theta$ et $b = \theta$ :
$\cos((n+1)\theta) + \cos((n-1)\theta) = 2\cos\theta \cos(n\theta)$. En posant $X = \cos\theta$, on a bien $T_{n+1} = 2XT_n - T_{n-1}$.
2. $T_n(x) = 0 \iff \cos(n\theta) = 0 \iff n\theta = \frac{\pi}{2} + k\pi$.
Les racines sont $x_k = \cos\left(\frac{2k+1}{2n}\pi\right)$ pour $k \in \{0, \dots, n-1\}$.
Comme $\frac{2k+1}{2n}\pi \in ]0, \pi[$, les cosinus sont tous dans $]-1, 1[$.

Visualisation : Notez que toutes les racines sont réelles et que le polynôme oscille entre -1 et 1.
Technique III. Racines et Multiplicité
Soit $P = nX^{n+1} - (n+1)X^n + 1$.
1. Montrer que 1 est racine de $P$.
2. Déterminer son ordre de multiplicité.
3. En déduire la factorisation de $P$ pour $n=2$.
Corrigé :
1. $P(1) = n(1)^{n+1} - (n+1)(1)^n + 1 = n - n - 1 + 1 = 0$.
2. Dérivons : $P' = n(n+1)X^n - n(n+1)X^{n-1}$.
$P'(1) = n(n+1) - n(n+1) = 0$.
$P'' = n^2(n+1)X^{n-1} - n(n+1)(n-1)X^{n-2}$.
$P''(1) = n^2(n+1) - n(n^2-1) = n^3 + n^2 - n^3 + n = n^2 + n \neq 0$.
La multiplicité est donc **exactement 2**.

Le graphe est tangent à l'axe des abscisses en $x=1$ sans le traverser (pour une racine double).
Expert IV. Racines de l'Unité & D.E.S.
Décomposer en éléments simples $F = \frac{1}{X^n-1}$ dans $\mathbb{C}(X)$.
Méthode des résidus :
Les pôles sont les $\omega_k = e^{i\frac{2k\pi}{n}}$. Ce sont des pôles simples.
La partie entière est nulle car $\deg(1) < \deg(X^n-1)$.
Le coefficient $\lambda_k$ associé au pôle $\omega_k$ est donné par $\frac{P(\omega_k)}{Q'(\omega_k)}$ :
$\lambda_k = \frac{1}{n\omega_k^{n-1}} = \frac{\omega_k}{n\omega_k^n} = \frac{\omega_k}{n}$ (car $\omega_k^n = 1$).
$F(X) = \frac{1}{n} \sum_{k=0}^{n-1} \frac{e^{i\frac{2k\pi}{n}}}{X - e^{i\frac{2k\pi}{n}}}$

Les pôles de la fraction sont les sommets d'un polygone régulier inscrit dans le cercle unité.
DS Type V. Arithmétique & Bézout
Trouver les polynômes $U$ et $V$ tels que $(X^3-1)U + (X^2+1)V = 1$.
Étapes de calcul :
1. $X^3-1 = X(X^2+1) + (-X-1)$
2. $X^2+1 = (-X+1)(-X-1) + 2$
On remonte :
$2 = (X^2+1) - (-X+1)(-X-1)$
$2 = (X^2+1) - (-X+1)[(X^3-1) - X(X^2+1)]$
En regroupant les termes :
$1 = \frac{1}{2}(X-1)(X^3-1) + \frac{1}{2}(-X^2+X+1)(X^2+1)$
