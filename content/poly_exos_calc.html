5 Exercices d'entraînement lourd (Type DS PCSI) avec corrigés hyper détaillés.

Exercice 1 : Division euclidienne et restes Classique
  1. Énoncer le théorème de la division euclidienne d'un polynôme $A$ par un polynôme $B$ ($B \neq 0$).
  2. Soit $A(X) = X^n$ (avec $n \geq 2$) et $B(X) = (X-a)(X-b)$ avec $a \neq b$. Quel est le degré maximal du reste $R$ ?
  3. Écrire la forme générale de $R(X)$.
  4. Évaluer la relation $A = BQ + R$ en $a$ et en $b$.
  5. En déduire l'expression explicite du reste $R(X)$ en fonction de $a, b$ et $n$.
  6. Que devient la méthode si $a = b$ (division par $(X-a)^2$) ? Expliquer comment utiliser la dérivée.
  7. Application : Déterminer le reste de la division euclidienne de $(X \sin \theta + \cos \theta)^n$ par $X^2+1$ dans $\mathbb{C}[X]$.

Corrigé de l'Exercice 1

  1. Il existe un unique couple de polynômes $(Q, R) \in \mathbb{K}[X]^2$ tel que :
    $A = BQ + R \quad \text{avec} \quad \deg(R) < \deg(B)$
    (On rappelle par convention que le polynôme nul a pour degré $-\infty$).
  2. Ici, $B(X) = (X-a)(X-b)$, donc $\deg(B) = 2$. Par conséquent, $\deg(R) < 2$, ce qui signifie que le degré maximal de $R$ est $1$.
  3. Puisque $\deg(R) \le 1$, on peut l'écrire sous la forme :
    $R(X) = \alpha X + \beta \quad \text{où} \quad (\alpha, \beta) \in \mathbb{K}^2$
  4. On part de l'égalité $X^n = (X-a)(X-b)Q(X) + \alpha X + \beta$.
    En évaluant en $X = a$, le terme en $B(X)Q(X)$ s'annule : $a^n = \alpha a + \beta$.
    En évaluant en $X = b$ : $b^n = \alpha b + \beta$.
  5. On doit résoudre le système d'équations :
    (1) $\alpha a + \beta = a^n$
    (2) $\alpha b + \beta = b^n$
    En soustrayant (2) à (1), on élimine $\beta$ : $\alpha(a - b) = a^n - b^n$. Puisque $a \neq b$, on divise par $a-b$ :
    $\alpha = \frac{a^n - b^n}{a-b}$.
    On remplace $\alpha$ dans (1) pour trouver $\beta$ :
    $\beta = a^n - a \left( \frac{a^n - b^n}{a-b} \right) = \frac{a^n(a-b) - a(a^n - b^n)}{a-b} = \frac{a^{n+1} - a^nb - a^{n+1} + ab^n}{a-b} = \frac{ab^n - ba^n}{a-b}$.
    Le reste est donc : $R(X) = \frac{a^n - b^n}{a-b} X + \frac{ab^n - ba^n}{a-b}$.
  6. Si $a = b$, la division se fait par $B(X) = (X-a)^2$. On pose $X^n = (X-a)^2 Q(X) + \alpha X + \beta$.
    L'évaluation en $a$ donne : $a^n = \alpha a + \beta$. (Équation 1)
    On dérive ensuite l'égalité $A = BQ + R$ :
    $n X^{n-1} = 2(X-a)Q(X) + (X-a)^2 Q'(X) + \alpha$.
    En évaluant cette dérivée en $X = a$, tous les termes contenant $(X-a)$ s'annulent, ce qui donne directement :
    $\alpha = n a^{n-1}$.
    On remplace $\alpha$ dans l'Équation 1 : $\beta = a^n - a(n a^{n-1}) = a^n - n a^n = (1-n)a^n$.
    Le reste est donc : $R(X) = n a^{n-1} X + (1-n)a^n$.
  7. On divise $A(X) = (X \sin \theta + \cos \theta)^n$ par $B(X) = X^2+1$. Les racines de $B$ sont complexes : $i$ et $-i$.
    On écrit $A(X) = (X^2+1)Q(X) + \alpha X + \beta$.
    On évalue en $X = i$ : $A(i) = (i \sin \theta + \cos \theta)^n$. D'après la formule d'Euler, $A(i) = (e^{i\theta})^n = e^{in\theta}$.
    Par la formule de Moivre, $e^{in\theta} = \cos(n\theta) + i\sin(n\theta)$.
    Du côté droit de l'égalité, $(i^2+1)Q(i) + \alpha i + \beta = \alpha i + \beta$.
    On identifie les parties réelle et imaginaire (puisque $\alpha, \beta \in \mathbb{R}$) :
    $\alpha = \sin(n\theta)$ et $\beta = \cos(n\theta)$.
    Le reste est donc : $R(X) = \sin(n\theta) X + \cos(n\theta)$.
Exercice 2 : Racines, multiplicité et dérivation Incontournable

Soit $n \in \mathbb{N}^*$. On considère le polynôme $P_n(X) = X^{2n} - nX^{n+1} + nX^{n-1} - 1$.

  1. Calculer $P_n(1)$. Que peut-on en déduire ?
  2. Calculer les polynômes dérivés $P_n'(X)$ et $P_n''(X)$.
  3. Évaluer $P_n'(1)$ et $P_n''(1)$.
  4. En déduire l'ordre de multiplicité exact de la racine $1$ pour le polynôme $P_n$.
  5. Le scalaire $-1$ est-il racine de $P_n$ ? Étudier selon la parité de $n$.
  6. Déterminer l'ordre de multiplicité de la racine $-1$ lorsqu'elle existe.
  7. Expliciter et factoriser complètement $P_2(X)$ dans $\mathbb{R}[X]$.

Corrigé de l'Exercice 2

  1. On remplace $X$ par $1$ : $P_n(1) = 1^{2n} - n(1^{n+1}) + n(1^{n-1}) - 1 = 1 - n + n - 1 = 0$.
    On en déduit que $1$ est une racine de $P_n$.
  2. On utilise les règles de dérivation usuelles ($d(x^k)/dx = k x^{k-1}$) :
    $P_n'(X) = 2n X^{2n-1} - n(n+1) X^n + n(n-1) X^{n-2}$

    $P_n''(X) = 2n(2n-1) X^{2n-2} - n^2(n+1) X^{n-1} + n(n-1)(n-2) X^{n-3}$
  3. On évalue la première dérivée en $1$ :
    $P_n'(1) = 2n(1) - n(n+1)(1) + n(n-1)(1) = 2n - n^2 - n + n^2 - n = 0$.
    On évalue la dérivée seconde en $1$ :
    $P_n''(1) = 2n(2n-1) - n^2(n+1) + n(n-1)(n-2)$
    $P_n''(1) = (4n^2 - 2n) - (n^3 + n^2) + (n^3 - 3n^2 + 2n)$
    $P_n''(1) = n^3 - n^3 + 4n^2 - n^2 - 3n^2 - 2n + 2n = 0$.
  4. D'après le cours, un scalaire $\alpha$ est racine d'ordre au moins $k$ si $P(\alpha) = P'(\alpha) = \dots = P^{(k-1)}(\alpha) = 0$.
    Ici, $P_n(1) = P_n'(1) = P_n''(1) = 0$, donc $1$ est racine de multiplicité au moins 3.
    Vérifions si la dérivée troisième s'annule :
    $P_n'''(X) = 2n(2n-1)(2n-2)X^{2n-3} - n^2(n^2-1)X^{n-2} + n(n-1)(n-2)(n-3)X^{n-4}$.
    Un calcul montre que $P_n'''(1) = n(n^2-1)$. Pour $n \ge 2$, $P_n'''(1) \neq 0$. La multiplicité est donc exactement 3.
  5. Évaluons $P_n(-1)$ :
    $P_n(-1) = (-1)^{2n} - n(-1)^{n+1} + n(-1)^{n-1} - 1$.
    Sachant que $(-1)^{2n} = 1$ et que $(-1)^{n+1} = (-1)^2(-1)^{n-1} = (-1)^{n-1}$ :
    $P_n(-1) = 1 - n(-1)^{n-1} + n(-1)^{n-1} - 1 = 0$.
    $-1$ est donc toujours racine, quelle que soit la parité de $n$.
  6. Évaluons $P_n'(-1)$ :
    $P_n'(-1) = 2n(-1)^{2n-1} - n(n+1)(-1)^n + n(n-1)(-1)^{n-2}$.
    Puisque $2n-1$ est toujours impair, $(-1)^{2n-1} = -1$. De plus, $(-1)^{n-2} = (-1)^n$.
    $P_n'(-1) = -2n - n(n+1)(-1)^n + n(n-1)(-1)^n = -2n + (-1)^n [ -n^2 - n + n^2 - n ]$
    $P_n'(-1) = -2n - 2n(-1)^n = -2n (1 + (-1)^n)$.
    - Si $n$ est pair : $(-1)^n = 1$, donc $P_n'(-1) = -4n \neq 0$. La racine est simple (multiplicité 1).
    - Si $n$ est impair : $(-1)^n = -1$, donc $P_n'(-1) = 0$. La racine est multiple (multiplicité $\ge 2$).
  7. Pour $n=2$ : $P_2(X) = X^4 - 2X^3 + 2X - 1$.
    D'après ce qui précède, pour $n=2$ (pair), $1$ est racine de multiplicité 3, et $-1$ est racine simple.
    La somme des multiplicités vaut $3 + 1 = 4$, ce qui correspond au degré du polynôme !
    Puisque le coefficient dominant est 1, on en déduit directement sa factorisation :
    $P_2(X) = (X-1)^3(X+1)$.
Exercice 3 : Polynômes de Tchebychev Difficile

On s'intéresse à la famille de polynômes $T_n$ définie par la propriété : $\forall \theta \in \mathbb{R}, T_n(\cos \theta) = \cos(n\theta)$.

  1. En utilisant la formule de Moivre, justifier l'existence d'un tel polynôme $T_n \in \mathbb{R}[X]$.
  2. Calculer explicitement $T_0(X), T_1(X)$ et $T_2(X)$.
  3. À l'aide des formules d'addition trigonométriques $\cos(a+b)+\cos(a-b)$, établir la relation : $T_{n+2}(X) = 2X T_{n+1}(X) - T_n(X)$.
  4. Déterminer par récurrence le degré et le coefficient dominant de $T_n$ (pour $n \geq 1$).
  5. Quelle est la parité du polynôme $T_n$ ?
  6. Résoudre l'équation $\cos(n\theta) = 0$ pour $\theta \in [0, \pi]$. En déduire les racines de $T_n$.
  7. Montrer que $T_n$ est scindé à racines simples dans $\mathbb{R}[X]$ et écrire sa factorisation.

Corrigé de l'Exercice 3

  1. La formule de Moivre donne $(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta)$.
    En développant le membre de gauche avec la formule du binôme de Newton, puis en identifiant les parties réelles :
    $\cos(n\theta) = \sum_{k=0}^{\lfloor n/2 \rfloor} \binom{n}{2k} \cos^{n-2k}(\theta) i^{2k} \sin^{2k}(\theta)$.
    Comme $i^{2k} = (-1)^k$ et $\sin^{2k}(\theta) = (\sin^2\theta)^k = (1-\cos^2\theta)^k$, tous les termes s'expriment comme des puissances de $\cos\theta$. En remplaçant $\cos\theta$ par $X$, on obtient bien une expression purement polynomiale à coefficients réels.
  2. On applique la définition pour les premières valeurs :
    - Pour $n=0$ : $\cos(0\theta) = 1$. Donc $T_0(\cos\theta) = 1 \implies$ $T_0(X) = 1$.
    - Pour $n=1$ : $\cos(1\theta) = \cos\theta$. Donc $T_1(\cos\theta) = \cos\theta \implies$ $T_1(X) = X$.
    - Pour $n=2$ : $\cos(2\theta) = 2\cos^2\theta - 1$. Donc $T_2(\cos\theta) = 2\cos^2\theta - 1 \implies$ $T_2(X) = 2X^2 - 1$.
  3. On utilise la formule trigonométrique $\cos(p) + \cos(q) = 2\cos\left(\frac{p+q}{2}\right)\cos\left(\frac{p-q}{2}\right)$.
    Posons $p = (n+2)\theta$ et $q = n\theta$.
    $\cos((n+2)\theta) + \cos(n\theta) = 2\cos\left(\frac{2n\theta + 2\theta}{2}\right)\cos\left(\frac{2\theta}{2}\right) = 2\cos((n+1)\theta)\cos(\theta)$.
    En réécrivant avec les polynômes de Tchebychev :
    $T_{n+2}(\cos\theta) + T_n(\cos\theta) = 2 \cos\theta \, T_{n+1}(\cos\theta)$.
    En posant $X = \cos\theta$ (qui balaye $[-1, 1]$, segment infini donc suffisant pour identifier des polynômes), on obtient bien : $T_{n+2}(X) = 2X T_{n+1}(X) - T_n(X)$.
  4. Montrons par récurrence double que pour $n \ge 1$, $\deg(T_n) = n$ et son coefficient dominant est $2^{n-1}$.
    - Initialisation : $T_1 = X$ (degré 1, dominant $2^0=1$) et $T_2 = 2X^2-1$ (degré 2, dominant $2^1=2$). C'est vrai.
    - Hérédité : Supposons la propriété vraie pour les rangs $n$ et $n+1$.
    $T_{n+2}(X) = 2X T_{n+1}(X) - T_n(X)$.
    Par hypothèse, $T_{n+1}$ est de degré $n+1$ avec dominant $2^n$. Donc $2X T_{n+1}(X)$ est de degré $n+2$ avec dominant $2 \times 2^n = 2^{n+1}$. Comme $\deg(T_n) = n < n+2$, le terme soustrait n'affecte pas le coefficient dominant. La propriété est donc vraie au rang $n+2$.
  5. On remarque que $T_0$ est pair, $T_1$ est impair, $T_2$ est pair.
    D'après la relation de récurrence $T_{n+2}(X) = 2X T_{n+1}(X) - T_n(X)$, multiplier par $X$ change la parité. On en déduit par récurrence que $T_n$ a la même parité que $n$ : $T_n(-X) = (-1)^n T_n(X)$.
  6. $\cos(n\theta) = 0 \iff n\theta \equiv \frac{\pi}{2} \pmod{\pi} \iff n\theta = \frac{\pi}{2} + k\pi \iff \theta = \frac{(2k+1)\pi}{2n}$, avec $k \in \mathbb{Z}$.
    Pour chercher les racines du polynôme, on restreint $\theta$ à l'intervalle $[0, \pi]$ (où le cosinus est bijectif). Cela correspond aux valeurs $k \in \{0, 1, \dots, n-1\}$.
    Les racines de $T_n$ sont donc les $x_k = \cos\left(\frac{(2k+1)\pi}{2n}\right)$ pour $k \in \{0, \dots, n-1\}$.
  7. Nous avons trouvé $n$ racines distinctes (car la fonction cosinus est strictement décroissante sur $[0, \pi]$ donc injective, les $n$ angles distincts donnent $n$ cosinus distincts).
    Comme $\deg(T_n) = n$, le polynôme possède au maximum $n$ racines. Puisqu'on en a trouvé $n$, il est scindé à racines simples.
    La factorisation s'écrit (sans oublier le coefficient dominant $2^{n-1}$) :
    $T_n(X) = 2^{n-1} \prod_{k=0}^{n-1} \left(X - \cos\left(\frac{(2k+1)\pi}{2n}\right)\right)$.
Exercice 4 : Bézout et Algorithme d'Euclide Technique

On donne les polynômes $C(X) = X^3 - 1$ et $D(X) = X^2 + 1$.

  1. Factoriser $C$ et $D$ dans $\mathbb{C}[X]$ puis dans $\mathbb{R}[X]$.
  2. Montrer, sans faire de division, que $C$ et $D$ sont premiers entre eux.
  3. Effectuer l'algorithme d'Euclide sur $C$ et $D$ pour trouver leur PGCD.
  4. Remonter l'algorithme d'Euclide pour expliciter un couple de polynômes $(U, V)$ tel que $C U + D V = 1$.
  5. On cherche à résoudre l'équation $(E) : C P + D Q = X^4$ avec $P, Q \in \mathbb{R}[X]$. Donner une solution particulière $(P_0, Q_0)$.
  6. Déterminer l'ensemble de toutes les solutions $(P, Q)$ de l'équation $(E)$.
  7. Montrer qu'il existe un unique couple $(P, Q)$ solution tel que $\deg(P) < \deg(D)$. Déterminer ce couple.

Corrigé de l'Exercice 4

  1. Factorisons $C(X) = X^3 - 1$. L'identité remarquable $a^3-b^3$ donne $(X-1)(X^2+X+1)$.
    - Dans $\mathbb{R}[X]$ : le discriminant de $X^2+X+1$ est $\Delta = -3 < 0$, donc irréductible sur $\mathbb{R}$.
    $C(X) = (X-1)(X^2+X+1)$ et $D(X) = X^2+1$.
    - Dans $\mathbb{C}[X]$ : les racines de $X^2+X+1$ sont les racines cubiques de l'unité complexes $j = e^{i2\pi/3}$ et $\bar{j} = j^2$. Les racines de $X^2+1$ sont $\pm i$.
    $C(X) = (X-1)(X-j)(X-j^2)$ et $D(X) = (X-i)(X+i)$.
  2. Pour qu'ils ne soient pas premiers entre eux, ils devraient partager une racine complexe. Les seules racines de $D$ sont $i$ et $-i$.
    On calcule $C(i) = i^3 - 1 = -i - 1 \neq 0$.
    On calcule $C(-i) = (-i)^3 - 1 = i - 1 \neq 0$.
    Puisqu'ils n'ont aucune racine en commun dans $\mathbb{C}$, ils n'ont aucun facteur commun : $PGCD(C,D)=1$.
  3. Première division (Euclide de $X^3-1$ par $X^2+1$) :
    $X^3 - 1 = (X^2+1) \times X - X - 1$.
    Le reste est $R_1(X) = -X-1$.
    Deuxième division (Euclide de $X^2+1$ par $-X-1$) :
    $X^2+1 = (-X-1)(-X+1) + 2$.
    Le reste est $R_2(X) = 2$. C'est une constante non nulle, le dernier reste non nul représente le PGCD. À un coefficient multiplicatif près, on retrouve bien PGCD = 1.
  4. On isole le reste constant dans la dernière équation :
    $2 = (X^2+1) - (-X-1)(-X+1)$
    On substitue $R_1(X) = -X-1$ par l'expression tirée de la première division : $-X-1 = (X^3-1) - X(X^2+1)$.
    $2 = (X^2+1) - [ (X^3-1) - X(X^2+1) ](-X+1)$
    On développe prudemment le membre de droite pour regrouper $C$ et $D$ :
    $2 = D - C(-X+1) + D \times X(-X+1)$
    $2 = D \cdot (1 - X^2 + X) + C \cdot (X-1)$.
    Pour obtenir $1$, on divise tout par 2 :
    $1 = C(X) \left[ \frac{X-1}{2} \right] + D(X) \left[ \frac{-X^2+X+1}{2} \right]$.
    Donc $U(X) = \frac{1}{2}(X-1)$ et $V(X) = \frac{1}{2}(-X^2+X+1)$.
  5. On a l'identité de Bézout $CU + DV = 1$. Il suffit de multiplier les deux membres par $X^4$ :
    $C(X) [X^4 U(X)] + D(X) [X^4 V(X)] = X^4$.
    Une solution particulière est donc :
    $P_0(X) = \frac{1}{2}(X^5-X^4)$ et $Q_0(X) = \frac{1}{2}(-X^6+X^5+X^4)$.
  6. Soit $(P, Q)$ une solution quelconque. L'équation est $CP + DQ = X^4$.
    En soustrayant $CP_0 + DQ_0 = X^4$, on obtient :
    $C(P - P_0) + D(Q - Q_0) = 0 \implies C(P - P_0) = -D(Q - Q_0)$.
    Cela signifie que $D$ divise le produit $C(P-P_0)$.
    Or, $PGCD(C,D) = 1$. D'après le Lemme de Gauss, $D$ divise $(P-P_0)$.
    Il existe donc un polynôme $K \in \mathbb{R}[X]$ tel que $P - P_0 = K \cdot D \implies P = P_0 + K \cdot D$.
    En réinjectant dans $C(K \cdot D) = -D(Q - Q_0)$, on simplifie par $D$ (non nul) : $Q - Q_0 = -K \cdot C \implies Q = Q_0 - K \cdot C$.
    L'ensemble des solutions est : $\mathcal{S} = \{ (P_0 + KD, Q_0 - KC) \mid K \in \mathbb{R}[X] \}$.
  7. Pour que $\deg(P) < \deg(D) = 2$, il faut chercher le reste de la division euclidienne de la solution générale par $D$.
    Posons la division euclidienne de $P_0$ par $D$ : $P_0 = D \cdot S + R$, avec $\deg(R) < 2$.
    En choisissant le polynôme arbitraire $K = -S$, la solution $P$ devient exactement $R$.
    Effectuons la division : $P_0(X) = \frac{1}{2}(X^5-X^4) = \frac{1}{2} (X^2+1)(X^3 - X^2 - X + 1) + \frac{1}{2}(-X-1)$.
    Le reste est donc le polynôme cherché : $P(X) = \frac{1}{2}(-X-1)$.
    Pour trouver $Q$, on utilise la relation $Q = \frac{X^4 - CP}{D}$ :
    $X^4 - CP = X^4 - (X^3-1)\frac{1}{2}(-X-1) = X^4 + \frac{1}{2}(X^4 + X^3 - X - 1) = \frac{1}{2}(2X^4 + X^4 + X^3 - X - 1) = \frac{1}{2}(3X^4 + X^3 - X - 1)$.
    Wait, recalculons proprement via les équations simplifiées pour éviter l'erreur. On peut juste poser $Q(X)$ de degré 2 (car $CP + DQ = X^4 \implies \deg(DQ) = 4 \implies \deg(Q) = 2$).
    $Q(X) = aX^2+bX+c$. $X^4 = (X^3-1)(-\frac{X+1}{2}) + (X^2+1)(aX^2+bX+c)$.
    $X^4 = -\frac{1}{2}X^4 - \frac{1}{2}X^3 + \frac{1}{2}X + \frac{1}{2} + aX^4 + bX^3 + (a+c)X^2 + bX + c$.
    Identification des coefficients :
    $X^4 : 1 = -\frac{1}{2} + a \implies a = \frac{3}{2}$.
    $X^3 : 0 = -\frac{1}{2} + b \implies b = \frac{1}{2}$.
    $X^2 : 0 = a + c \implies c = -a = -\frac{3}{2}$.
    Donc $Q(X) = \frac{1}{2}(3X^2 + X - 3)$.
Exercice 5 : Décomposition en Éléments Simples (D.E.S) Expert

Soit $n \in \mathbb{N}^*$. On étudie la fraction rationnelle $F_n(X) = \frac{1}{X^n - 1}$ dans $\mathbb{C}(X)$.

  1. Déterminer les pôles de $F_n$. On notera $\omega_k = e^{i 2k\pi / n}$ pour $k \in \{0, \dots, n-1\}$.
  2. Justifier que tous ces pôles sont simples et donner la forme théorique de la D.E.S de $F_n$ dans $\mathbb{C}(X)$.
  3. Rappeler la formule de cours permettant de trouver le coefficient $\lambda_k$ associé à un pôle simple $a$ d'une fraction $P/Q$.
  4. Appliquer cette formule pour déterminer $\lambda_k$ en fonction de $\omega_k$ et de $n$.
  5. En déduire l'écriture complète de $F_n(X)$ sous forme de somme.
  6. En multipliant l'égalité par $X$ et en faisant tendre $|X| \to +\infty$, que vaut la somme $\sum_{k=0}^{n-1} \lambda_k$ ?
  7. Bonus : Déterminer la valeur de $\sum_{k=0}^{n-1} \omega_k$ à partir des propriétés du polynôme $X^n-1$.

Corrigé de l'Exercice 5

  1. Les pôles d'une fraction sous forme irréductible sont les racines de son dénominateur. Ici, on cherche les solutions de $X^n - 1 = 0 \iff X^n = 1$.
    Ce sont exactement les racines $n$-ièmes de l'unité : $\omega_k = e^{i 2k\pi / n}$ pour $k \in \{0, \dots, n-1\}$.
  2. Notons $Q(X) = X^n-1$. Pour vérifier la multiplicité d'une racine, on regarde si elle annule la dérivée. $Q'(X) = nX^{n-1}$.
    $Q'(\omega_k) = n(\omega_k)^{n-1} \neq 0$ car $\omega_k \neq 0$. Les $n$ pôles ne s'annulent pas dans la dérivée, ils sont donc tous simples.
    Puisque $\deg(numérateur) = 0$ et $\deg(dénominateur) = n \ge 1$, la partie entière est nulle. La décomposition théorique est une somme de fractions de degré 1 :
    $F_n(X) = \sum_{k=0}^{n-1} \frac{\lambda_k}{X-\omega_k}$.
  3. D'après le cours, si une fraction rationnelle s'écrit $F = \frac{P}{Q}$ et si $a$ est un pôle simple, le coefficient (ou résidu) associé à l'élément simple $\frac{\lambda}{X-a}$ se calcule via :
    $\lambda = \frac{P(a)}{Q'(a)}$.
  4. Ici, $P(X) = 1$ et $Q'(X) = nX^{n-1}$. Pour le pôle $\omega_k$ :
    $\lambda_k = \frac{1}{n \omega_k^{n-1}}$.
    Or, on sait que $\omega_k^n = 1$ (c'est une racine $n$-ième de l'unité). On peut donc réécrire :
    $\omega_k^{n-1} = \frac{\omega_k^n}{\omega_k} = \frac{1}{\omega_k}$.
    En remplaçant dans l'expression de $\lambda_k$ :
    $\lambda_k = \frac{1}{n \left(\frac{1}{\omega_k}\right)} \implies$ $\lambda_k = \frac{\omega_k}{n}$.
  5. En réinjectant les coefficients trouvés dans la forme théorique, on obtient :
    $F_n(X) = \frac{1}{n} \sum_{k=0}^{n-1} \frac{\omega_k}{X-\omega_k}$.
  6. On multiplie notre fraction par $X$ :
    $X F_n(X) = \frac{X}{X^n-1} = \sum_{k=0}^{n-1} \frac{\lambda_k X}{X-\omega_k}$.
    Regardons la limite de chaque membre quand $X \to +\infty$ (en supposant $n \ge 2$) :
    - Pour le membre de gauche : $\frac{X}{X^n-1} \sim \frac{X}{X^n} = \frac{1}{X^{n-1}}$, qui tend vers $0$.
    - Pour le membre de droite : chaque terme $\frac{\lambda_k X}{X-\omega_k} = \lambda_k \frac{X}{X-\omega_k}$ tend vers $\lambda_k \times 1 = \lambda_k$. La somme tend donc vers $\sum_{k=0}^{n-1} \lambda_k$.
    Par identification des limites, on conclut que $\sum_{k=0}^{n-1} \lambda_k = 0$.
  7. D'après la question 4, on sait que $\lambda_k = \frac{\omega_k}{n}$. La somme de la question 6 s'écrit donc :
    $\sum_{k=0}^{n-1} \frac{\omega_k}{n} = 0 \implies \frac{1}{n} \sum_{k=0}^{n-1} \omega_k = 0 \implies$ $\sum_{k=0}^{n-1} \omega_k = 0$.
    (On retrouve ce résultat classique via les relations de Viète : la somme des racines d'un polynôme $X^n + a_{n-1}X^{n-1} + \dots + a_0$ est égale à $-a_{n-1}$. Dans $X^n-1$, le coefficient de $X^{n-1}$ est $0$).
