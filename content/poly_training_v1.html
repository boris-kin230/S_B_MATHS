Polynômes : Master Class V1

25 défis de niveau PCSI/MPSI. Maîtrisez les racines, la divisibilité et les relations de Viète.

Quel est le reste de la division euclidienne de $X^4 + X^2 + 1$ par $X^2 - X + 1$ ?
Correction : Remarquons que $(X^2 - X + 1)(X^2 + X + 1) = X^4 + X^2 + 1$. Le polynôme est donc un multiple, le reste est nul.
Soit $P(X) = 2X^3 - 4X^2 + 6X - 8$. Que vaut la somme de ses racines complexes ?
Correction : Relations de Viète : $\sum x_i = -a_{n-1}/a_n = -(-4)/2 = 2$.
Trouver $a$ et $b$ pour que $X^3 + aX + b$ soit divisible par $(X-1)^2$.
Correction : $P(1)=0$ et $P'(1)=0$. $1+a+b=0$ et $3+a=0 \Rightarrow a=-3, b=2$.
Degré de $P(X) = (X^2+1)^n - (X^2-1)^n$ pour $n \geq 1$ ?
Correction : Les termes en $X^{2n}$ s'annulent. Le terme dominant est $nX^{2n-2} - (-nX^{2n-2}) = 2nX^{2n-2}$.
Si $\forall x \in \mathbb{R}, P(\sin x) = 0$, alors :
Correction : Sinus prend une infinité de valeurs dans $[-1,1]$. Un polynôme ayant une infinité de racines est nul.
