Fractions Rationnelles & D.E.S

Pôles, partie entière et Décomposition en Éléments Simples.

1. Définitions et Pôles

Une fraction rationnelle $F \in \mathbb{K}(X)$ est un "quotient formel" de deux polynômes $P$ et $Q$ avec $Q \neq 0$ :

$$F = \frac{P}{Q}$$

On peut toujours supposer $F$ sous forme irréductible (c'est-à-dire que $P$ et $Q$ sont premiers entre eux).

  • Degré : $\deg(F) = \deg(P) - \deg(Q)$
  • Pôles : Les racines complexes du dénominateur $Q$ (pour la forme irréductible).
  • Zéros : Les racines complexes du numérateur $P$.

Application Pratique

Problème : Déterminer les pôles, les zéros et le degré de la fraction $F(X) = \frac{X^3 - 1}{X^2 - 3X + 2}$.

Solution : Attention, il faut d'abord s'assurer que la fraction est irréductible.

  • Factorisons le numérateur : $X^3 - 1 = (X - 1)(X^2 + X + 1)$.
  • Factorisons le dénominateur (racines évidente 1 et 2) : $X^2 - 3X + 2 = (X - 1)(X - 2)$.

On simplifie par $(X-1)$ : $F(X) = \frac{X^2 + X + 1}{X - 2}$.

Maintenant, on peut conclure : Le seul pôle est $2$ (de multiplicité 1). Les zéros sont les racines de $X^2+X+1$ (qui sont $j$ et $\bar{j}$ dans $\mathbb{C}$). Le degré est $\deg(F) = 2 - 1 = 1$.

2. Partie Entière

Toute fraction rationnelle $F = \frac{P}{Q}$ se décompose de manière unique sous la forme :

$$F = E + \frac{R}{Q}$$

Où $E$ est un polynôme appelé partie entière de $F$, et $\deg(R) < \deg(Q)$.

Astuce : $E$ est simplement le quotient de la division euclidienne de $P$ par $Q$. Si $\deg(P) < \deg(Q)$, alors la partie entière est nulle ($E=0$).

Démonstration de l'existence et de l'unicité

Existence : Effectuons la division euclidienne du polynôme $P$ par le polynôme $Q$ ($Q \neq 0$). Le théorème de la division euclidienne assure l'existence de deux polynômes $E$ (le quotient) et $R$ (le reste) tels que :

$$P = E \cdot Q + R \quad \text{avec} \quad \deg(R) < \deg(Q)$$

En divisant cette égalité par $Q$, on obtient directement $F = \frac{P}{Q} = E + \frac{R}{Q}$.

Unicité : Supposons qu'il existe une autre décomposition $F = E_1 + \frac{R_1}{Q}$. Alors $E + \frac{R}{Q} = E_1 + \frac{R_1}{Q}$, ce qui donne $E - E_1 = \frac{R_1 - R}{Q}$.

Si $E \neq E_1$, le degré du membre de gauche est $\ge 0$. Or, le membre de droite est une fraction dont le degré du numérateur est strictement inférieur à celui du dénominateur, donc son degré est $< 0$. C'est une contradiction. Donc $E = E_1$, ce qui implique $R = R_1$.

Application Pratique

Problème : Trouver la partie entière de $F(X) = \frac{X^3 + 2X + 1}{X^2 + 1}$.

Solution : On pose la division euclidienne de $X^3 + 2X + 1$ par $X^2 + 1$ :

$$X^3 + 2X + 1 = X(X^2 + 1) + X + 1$$

Ici, le quotient est $E(X) = X$ et le reste est $R(X) = X + 1$. La décomposition est donc :

$$F(X) = X + \frac{X + 1}{X^2 + 1}$$

La partie entière est $E(X) = X$.

3. Décomposition en Éléments Simples (D.E.S)

Théorème d'existence et d'unicité

Toute fraction rationnelle $F = \frac{P}{Q}$ s'écrit de manière unique comme la somme de sa partie entière et d'éléments simples associés à chaque pôle de $Q$.

Dans $\mathbb{C}(X)$ :

Les éléments simples sont tous de la forme $\frac{\lambda}{(X-a)^k}$. Exemple pour un pôle $a$ de multiplicité 2 :

$$\frac{c_1}{X-a} + \frac{c_2}{(X-a)^2}$$

Dans $\mathbb{R}(X)$ :

S'ajoutent des éléments simples de seconde espèce pour les polynômes irréductibles de degré 2 ($\Delta < 0$) :

$$\frac{\alpha X + \beta}{(X^2+pX+q)^k}$$

Application Pratique

Problème : Donner la forme théorique de la D.E.S dans $\mathbb{R}(X)$ de $F(X) = \frac{1}{X(X-1)^2(X^2+1)}$.

Solution : Le degré du numérateur (0) est inférieur à celui du dénominateur (5), donc la partie entière est nulle. Les pôles réels sont 0 (simple) et 1 (double). Le terme $X^2+1$ est irréductible dans $\mathbb{R}$. La forme théorique est :

$$F(X) = \frac{a}{X} + \frac{b}{X-1} + \frac{c}{(X-1)^2} + \frac{dX + e}{X^2+1}$$

Il "suffit" ensuite de trouver les constantes réelles $a, b, c, d, e$.

4. Astuces de calcul des coefficients

  • Pour un pôle simple $a$ : Le coefficient (résidu) s'obtient en calculant $\left[ (X-a)F(X) \right]_{X=a}$ ou bien avec la formule de Taylor : $\frac{P(a)}{Q'(a)}$.
  • Parité : Si $F$ est paire ou impaire, la D.E.S conserve les symétries, ce qui réduit de moitié les coefficients à calculer.
  • Limite à l'infini : Multiplier l'égalité de la D.E.S par $X$ et faire tendre $X \to +\infty$ donne une relation simple sur la somme des coefficients de degré 1.

Application Pratique (Pôle simple et Limite)

Problème : Décomposer $F(X) = \frac{X}{(X-1)(X+2)}$.

Solution : La forme théorique est $F(X) = \frac{a}{X-1} + \frac{b}{X+2}$.

  • Calcul de $a$ : On multiplie $F(X)$ par $(X-1)$ et on évalue en $X=1$ :
    $a = \left[ \frac{X}{X+2} \right]_{X=1} = \frac{1}{3}$.
  • Calcul de $b$ (par la limite) : On multiplie tout par $X$ :
    $X F(X) = \frac{aX}{X-1} + \frac{bX}{X+2}$.
    En faisant tendre $X \to +\infty$, $\frac{X^2}{(X-1)(X+2)} \to 1$. À droite, l'expression tend vers $a + b$. Donc $1 = a + b$.
    Comme $a = \frac{1}{3}$, on en déduit $b = 1 - \frac{1}{3} = \frac{2}{3}$.

Conclusion : $\frac{X}{(X-1)(X+2)} = \frac{1/3}{X-1} + \frac{2/3}{X+2}$.

5. Astuce pour un pôle multiple

Lorsqu'on a un pôle $a$ de multiplicité $k > 1$, la méthode classique de la multiplication fonctionne pour le coefficient de plus haut degré, mais pas pour les autres.

Pour la partie polaire $\frac{c_1}{X-a} + \frac{c_2}{(X-a)^2} + \dots + \frac{c_k}{(X-a)^k}$ :

  • Obtenir $c_k$ : On multiplie par $(X-a)^k$ et on évalue en $a$.
  • Obtenir les autres coefficients : On peut poser $X = a + h$ (ce qui ramène le pôle en 0) et faire un développement limité, ou bien faire la soustraction $F(X) - \frac{c_k}{(X-a)^k}$ et simplifier.

6. Application : Calcul de Primitives

La D.E.S est l'outil principal pour intégrer les fonctions rationnelles. Une fois décomposée, chaque élément simple possède une primitive connue :

  • $\int \frac{a}{x - \lambda} dx = a \ln|x - \lambda| + C$
  • $\int \frac{a}{(x - \lambda)^k} dx = \frac{a}{(1-k)(x - \lambda)^{k-1}} + C \quad (\text{pour } k \ge 2)$
  • Les éléments de seconde espèce $\frac{\alpha x + \beta}{x^2+px+q}$ donnent naissance à des termes en $\ln(x^2+px+q)$ et en $\arctan(\dots)$.

Exemple : Calculer $\int \frac{1}{x^2-1} dx$.

On décompose : $\frac{1}{x^2-1} = \frac{1}{(x-1)(x+1)} = \frac{1/2}{x-1} - \frac{1/2}{x+1}$.

On intègre : $\int \frac{1}{x^2-1} dx = \frac{1}{2} \ln|x-1| - \frac{1}{2} \ln|x+1| + C = \frac{1}{2} \ln \left| \frac{x-1}{x+1} \right| + C$.
