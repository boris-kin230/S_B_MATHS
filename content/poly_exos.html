Exercice 1 : Polynômes de Tchebychev $T_n$

Énoncé :
1. Justifier l'existence de $T_n$ tel que $T_n(\cos \theta) = \cos(n\theta)$.
2. Calculer $T_0, T_1, T_2, T_3$.
3. Démontrer la relation $T_{n+2} = 2XT_{n+1} - T_n$.
4. Déterminer le degré et le coefficient dominant de $T_n$ pour $n \ge 1$.
5. Calculer $T_n(1)$ et $T_n(-1)$.
6. Déterminer les racines de $T_n$ sur $[-1, 1]$.
7. Montrer que $T_n$ est pair si $n$ est pair, et impair si $n$ est impair.
8. Établir que pour $x \in [-1, 1]$, $|T_n(x)| \le 1$.
1. Existence : $\cos(n\theta) = \text{Re}(e^{in\theta}) = \text{Re}((\cos \theta + i \sin \theta)^n)$.
D'après le binôme de Newton : $\cos(n\theta) = \sum_{k=0, k \text{ pair}}^n \binom{n}{k} (\cos \theta)^{n-k} (i \sin \theta)^k$.
Comme $k=2p$, $(i \sin \theta)^{2p} = (-1)^p (\sin^2 \theta)^p = (-1)^p (1-\cos^2 \theta)^p$.
C'est une somme de puissances de $\cos \theta$, d'où l'existence de $T_n$.

2. Calculs :
$T_0 = 1$ ; $T_1 = X$ ; $T_2 = 2X^2-1$ ; $T_3 = 4X^3-3X$.

3. Récurrence : $\cos((n+2)\theta) + \cos(n\theta) = 2\cos((n+1)\theta)\cos\theta$.
En posant $X = \cos \theta$, on a $T_{n+2}(X) + T_n(X) = 2X T_{n+1}(X)$.

4. Degré/Coef : Par récurrence, $\deg(T_n)=n$.
$\text{dom}(T_{n+2}) = 2 \cdot \text{dom}(T_{n+1})$. Comme $\text{dom}(T_1)=1$, on a $\text{dom}(T_n) = 2^{n-1}$ pour $n \ge 1$.

6. Racines : $T_n(\cos \theta) = 0 \iff n\theta \equiv \frac{\pi}{2} [\pi]$.
$\theta_k = \frac{\pi}{2n} + \frac{k\pi}{n}$ pour $k \in \{0, \dots, n-1\}$.
$x_k = \cos\left(\frac{(2k+1)\pi}{2n}\right)$.

Exercice 2 : Polynômes de Legendre $L_n$

Énoncé : On pose $A_n = (X^2-1)^n$ et $L_n = \frac{1}{2^n n!} A_n^{(n)}$.
1. Préciser le degré de $L_n$.
2. Calculer $L_0$ et $L_1$.
3. Quelles sont les multiplicités des racines $1$ et $-1$ dans $A_n$ ?
4. Montrer par Rolle que $L_n$ possède $n$ racines distinctes dans $]-1, 1[$.
5. Calculer le coefficient dominant de $L_n$.
6. Montrer que $L_n(1) = 1$ via la formule de Leibniz.
7. Démontrer que $\int_{-1}^1 L_n(t) P(t) dt = 0$ pour tout $P \in \mathbb{R}_{n-1}[X]$.
8. Calculer $L_n(-1)$ en utilisant la parité.
3. Multiplicités : $A_n = (X-1)^n (X+1)^n$. Les racines $1$ et $-1$ sont de multiplicité $n$.

4. Rolle : $A_n(-1)=A_n(1)=0$. Par Rolle, $A_n'$ s'annule en $c_1 \in ]-1, 1[$.
Comme $-1$ et $1$ sont racines de $A_n'$ (multiplicité $n-1$), on réapplique Rolle sur $[-1, c_1]$ et $[c_1, 1]$.
Par itération, la dérivée $n$-ième possède $n$ racines distinctes dans $]-1, 1[$.

6. $L_n(1)$ : $A_n^{(n)} = [(X-1)^n (X+1)^n]^{(n)}$.
Leibniz : $\sum_{k=0}^n \binom{n}{k} ((X-1)^n)^{(k)} ((X+1)^n)^{(n-k)}$.
En $X=1$, seul le terme $k=n$ survit : $\binom{n}{n} n! (1+1)^n = n! 2^n$.
$L_n(1) = \frac{1}{2^n n!} (n! 2^n) = 1$.

7. Orthogonalité : Intégration par parties $n$ fois.
$\int_{-1}^1 A_n^{(n)}(t) P(t) dt = [A_n^{(n-1)}P] - \int A_n^{(n-1)}P'$.
Le crochet est nul car $A_n^{(k)}(\pm 1) = 0$ pour $k < n$.
Au bout de $n$ fois, on a $(-1)^n \int A_n P^{(n)}$. Or $\deg(P) < n \implies P^{(n)}=0$.

Exercice 3 : Décomposition et Sommes

Énoncé :
1. Décomposer $F = \frac{1}{X(X+1)}$ en éléments simples.
2. Calculer $S_n = \sum_{k=1}^n \frac{1}{k(k+1)}$.
3. Décomposer $G = \frac{1}{X(X+1)(X+2)}$.
4. Calculer $\sum_{k=1}^n \frac{1}{k(k+1)(k+2)}$.
5. Déterminer la partie entière de $H = \frac{X^3}{X^2-1}$.
6. Décomposer $H$ dans $\mathbb{R}(X)$.
7. Calculer l'intégrale $\int_2^3 \frac{t^3}{t^2-1} dt$.
8. Décomposer $\frac{1}{X^n-1}$ dans $\mathbb{C}(X)$.
1. $F$ : $F = \frac{1}{X} - \frac{1}{X+1}$.

2. $S_n$ : Somme télescopique : $(1 - 1/2) + (1/2 - 1/3) + \dots + (1/n - 1/(n+1)) = 1 - \frac{1}{n+1}$.

3. $G$ : $G = \frac{a}{X} + \frac{b}{X+1} + \frac{c}{X+2}$.
$a = [XG]_{0} = 1/2$. $b = [(X+1)G]_{-1} = -1$. $c = [(X+2)G]_{-2} = 1/2$.
$G = \frac{1}{2X} - \frac{1}{X+1} + \frac{1}{2(X+2)}$.

5. Partie entière : Division euclidienne de $X^3$ par $X^2-1$.
$X^3 = X(X^2-1) + X \implies H = X + \frac{X}{X^2-1}$. Partie entière = $X$.

8. $X^n-1$ : Les pôles sont $\omega_k = e^{2ik\pi/n}$.
$\frac{1}{X^n-1} = \sum_{k=0}^{n-1} \frac{\lambda_k}{X-\omega_k}$ avec $\lambda_k = \frac{1}{P'(\omega_k)} = \frac{1}{n\omega_k^{n-1}} = \frac{\omega_k}{n}$.

Exercice 4 : Polynômes et Matrices

Énoncé : Soit $A = \begin{pmatrix} 1 & 1 \\ 0 & 2 \end{pmatrix}$ et $P = (X-1)(X-2)$.
1. Calculer $A^2$.
2. Vérifier que $P(A) = 0_2$.
3. On pose $X^n = P(X)Q(X) + a_n X + b_n$. Écrire le système vérifié par $(a_n, b_n)$.
4. Résoudre le système pour exprimer $a_n$ et $b_n$ en fonction de $n$.
5. En déduire l'expression de $A^n$ en fonction de $A, I, n$.
6. Calculer les coefficients de $A^n$.
7. Soit $S_n = \sum_{k=0}^n A^k$. Exprimer $S_n$ à l'aide de $A^n$.
8. Calculer $(A-I)(A-2I)$. Conclure sur les valeurs propres.
3/4. Reste : $X^n = Q(X)(X-1)(X-2) + a_n X + b_n$.
En $X=1$ : $1 = a_n + b_n$.
En $X=2$ : $2^n = 2a_n + b_n$.
Soustraction : $a_n = 2^n - 1$.
Puis $b_n = 1 - (2^n - 1) = 2 - 2^n$.

5. $A^n$ : $A^n = (2^n-1)A + (2-2^n)I$.

6. Coefficients :
$A^n = (2^n-1)\begin{pmatrix} 1 & 1 \\ 0 & 2 \end{pmatrix} + \begin{pmatrix} 2-2^n & 0 \\ 0 & 2-2^n \end{pmatrix} = \begin{pmatrix} 1 & 2^n-1 \\ 0 & 2^n \end{pmatrix}$.

Exercice 5 : Synthèse et Trigonométrie

Énoncé : Soit $P = \sum_{k=0}^{n-1} X^k$.
1. Simplifier $P(X)$ pour $X \neq 1$.
2. Quelles sont les racines de $P$ dans $\mathbb{C}$ ?
3. Factoriser $P$ dans $\mathbb{C}[X]$.
4. En évaluant $P(1)$, montrer que $\prod_{k=1}^{n-1} (1 - e^{2ik\pi/n}) = n$.
5. Utiliser la factorisation de $1-e^{i\theta}$ pour faire apparaître un sinus.
6. En déduire que $\prod_{k=1}^{n-1} \sin\left(\frac{k\pi}{n}\right) = \frac{n}{2^{n-1}}$.
7. Soit $S = \sum_{k=0}^{n-1} \cos(2k\pi/n)$. Calculer $S$ pour $n \ge 2$.
8. Calculer $\sum_{k=0}^{n-1} \sin(2k\pi/n)$.
4. Produit : $P(1) = 1+1+\dots+1 = n$.
D'autre part $P(X) = \prod_{k=1}^{n-1} (X - \omega^k)$, donc $P(1) = \prod_{k=1}^{n-1} (1 - \omega^k) = n$.

5/6. Sinus : $1-e^{i\theta} = e^{i\theta/2}(e^{-i\theta/2}-e^{i\theta/2}) = -2ie^{i\theta/2}\sin(\theta/2)$.
En prenant le module : $|1-e^{i\theta}| = 2\sin(\theta/2)$ car $\theta/2 \in [0, \pi]$.
$|n| = \prod_{k=1}^{n-1} 2\sin(k\pi/n) \implies n = 2^{n-1} \prod \sin(k\pi/n)$.

7. Somme : $S = \text{Re}(\sum \omega^k)$.
Somme géométrique : $\sum_{k=0}^{n-1} \omega^k = \frac{\omega^n-1}{\omega-1} = \frac{1-1}{\omega-1} = 0$.
Donc $S = 0$.
