Exercices : Polynômes

Focus : Polynômes de Tchebychev ($T_n$)

Problème Classique : Famille de Tchebychev
On cherche une suite de polynômes $(T_n)$ tels que : $\forall \theta \in \mathbb{R}, \quad T_n(\cos \theta) = \cos(n\theta)$
  1. Démontrer l'unicité de $T_n$.
  2. Établir la relation de récurrence entre $T_n, T_{n-1}$ et $T_{n-2}$.
  3. Calculer les premiers termes.

1. Unicité

Si $P$ et $Q$ conviennent, alors $H = P-Q$ s'annule sur tout $[-1, 1]$. Un polynôme ayant une infinité de racines est nul, donc $P=Q$.

2. Récurrence

D'après la formule $2\cos a \cos b = \cos(a+b) + \cos(a-b)$ : $2\cos\theta \cos((n-1)\theta) = \cos(n\theta) + \cos((n-2)\theta)$ D'où : $\boxed{T_n(X) = 2XT_{n-1}(X) - T_{n-2}(X)}$

3. Premiers Termes

  • $T_0 = 1$
  • $T_1 = X$
  • $T_2 = 2X^2 - 1$
  • $T_3 = 4X^3 - 3X$

4. Racines de $T_n$

$T_n(\cos\theta) = 0 \iff n\theta = \frac{\pi}{2} + k\pi$. Les $n$ racines sont : $x_k = \cos\left( \frac{(2k+1)\pi}{2n} \right)$ pour $k \in \{0, \dots, n-1\}$.
